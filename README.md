# React Notes App

A simple notes application built with React.js.

## Features
- Add notes
- Delete notes
- Two-way form binding
- Dynamic rendering using map()
- useState hook
- Responsive UI with Tailwind CSS

## Technologies Used
- React.js
- Tailwind CSS
- JavaScript

## Learning Concepts
- Controlled Components
- State Management
- Event Handling
- Array Manipulation
- Conditional Rendering

## Installation

```bash
npm install
npm run dev
```
